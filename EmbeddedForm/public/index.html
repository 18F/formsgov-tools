This is the landing page for the embedded form testing project

This project tests embedding form.io forms different ways:

  1. Using Inline embedding
  2. Using Javascript embedding

It also contains simple example embedding pages that are similar to how certain projects are embedded in their calling applications. This is useful for testing, when wishing to simulate the calling environment as closely as possible:

  1. Made In America

    • Nonavailability Waiver
    • Financial Assistance Waiver
    • Urgent Waiver (TBD)

  2. Hud 903 Form (javascript embedding with submission page redirect)

    Ultimately the team determined to go with an application to host the HUD form.
    The application can be found here: https://github.com/18F/formservice-hud903
